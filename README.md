\# JAVA-POO



Laboratoire pédagogique Java orienté objet.

Ce dépôt regroupe des exercices et démonstrations sur :



\- Interfaces

\- Classes abstraites

\- Polymorphisme

\- Static vs non-static

\- Bonnes pratiques POO



\## Structure



src/

&nbsp;└── main/java/

&nbsp;    ├── InterfaceDemo.java

&nbsp;    ├── InterfaceeETabstraite.java

&nbsp;    ├── ClasseAbstraite.java

&nbsp;    ├── ClassAbsctac.java

&nbsp;    ├── StaticouNonStatic.java

&nbsp;    └── org/example/Polymorphisme.java



\## Prérequis



\- Java JDK 17+ (OpenJDK recommandé)

\- Maven

\- IntelliJ IDEA (optionnel)



\## Lancer un exemple



Depuis la racine du projet :



mvn clean compile

mvn exec:java



Ou directement depuis IntelliJ.



\## Objectif pédagogique



Ce projet sert de base de révision et de consolidation des concepts POO en Java,

avec une approche progressive et expérimentale.
